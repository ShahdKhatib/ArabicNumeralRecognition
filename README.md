# Arabic Numeral Recognition (0–9)

This project focuses on recognizing handwritten Arabic numerals (0–9) using both **classical machine learning techniques** and **deep learning models**. The goal is to compare different approaches and evaluate their performance on the same dataset.

---

##  Overview

Handwritten digit recognition is a fundamental problem in computer vision and pattern recognition. In this notebook, we explore:

* Data preprocessing and visualization
* Feature extraction techniques
* Classical machine learning models
* Deep learning models (neural networks)
* Performance comparison between approaches

---

## Methods Used

###  Classical Machine Learning

* K-Nearest Neighbors (KNN)
* Support Vector Machines (SVM)
* Logistic Regression

### 🔹 Deep Learning

* Fully Connected Neural Networks (ANN)
* (Optional if used: Convolutional Neural Networks - CNN)

---

##  Dataset

The dataset consists of handwritten Arabic numerals (0–9).
Each sample represents an image of a digit along with its corresponding label.

---

##  Workflow

1. **Data Loading**
2. **Preprocessing**

   * Normalization
   * Reshaping images
3. **Exploratory Data Analysis (EDA)**
4. **Model Training**
5. **Evaluation & Comparison**

##  Results

The performance of classical models is compared with deep learning models using accuracy and other evaluation metrics.

> Deep learning models generally outperform classical approaches due to their ability to automatically learn complex patterns from image data.
